# Machine Learning From Scratch

Hi, I’m Krish Verma 👋

This repository contains my implementations of some core Machine Learning algorithms built from scratch using Python.

I started this project because honestly, I was not fully understanding how ML models work internally. Using libraries is easy, but I wanted to know what’s actually happening behind the scenes. So I tried building everything step by step on my own.

# What I have implemented
Gradient Descent
Linear Regression
Multiple Linear Regression
Polynomial Regression
Ridge Regression
Lasso Regression
# What I learned

While working on these, I understood a lot of things more clearly:

How gradient descent actually updates values step by step
How cost function is minimized
Difference between linear, multiple and polynomial regression
How model becomes complex with polynomial features
How Ridge and Lasso help to reduce overfitting
Why regularization is important

At first, I was really confused about gradient descent and regularization, but after coding it myself, things started making sense.
